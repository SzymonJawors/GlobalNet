##  About the Project (EN)

**GlobalNet** is an interactive social media application built as a **Full-Stack (MERN)** project, designed to connect users globally. 

###  Key Features (Planned & Current)

####  For Users:
* **User Profile:** Full profile customization including cover photos and avatars.
* **Content Creation:** Create text posts and share multimedia content.
* **Interactions:** Like syste
* **Social Feed:** Dynamic feed displaying posts from friends and recommended content.
* **Friends System:** Send friend requests and manage connections.

####  Tech & Architecture:
* **State Management:** Advanced state handling using Redux Toolkit or Context API.
* **Responsiveness:** Fully responsive design for mobile and desktop (RWD).
* **Optimization:** Fast loading times powered by Vite.
